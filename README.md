# Phase 1 – Basic Expense Tracker

This phase implements a basic user interface for an expense tracker using:
- HTML
- CSS
- Basic JavaScript

Features:
- Simple expense input
- Display expenses in a list
- Beginner-friendly code with comments

# Phase 2 – Expense Tracking

This phase extends Phase 1 by adding proper expense tracking features.

Features:
Expense amount
Expense category
Expense date
Optional description
Expenses stored using JavaScript arrays and objects
Display of all recorded expenses
